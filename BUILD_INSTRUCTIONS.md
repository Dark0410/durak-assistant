# Инструкция по сборке APK

Для того чтобы получить готовый APK файл, выполните следующие шаги:

1.  **Установите Android Studio**: Скачайте и установите актуальную версию с официального сайта.
2.  **Откройте проект**: Запустите Android Studio и выберите "Open", затем укажите папку `c:\Open_Folder_Cod\DURAK`.
3.  **Дождитесь синхронизации Gradle**: Студия автоматически скачает необходимые зависимости (OpenCV, Retrofit и др.).
4.  **Сборка APK**:
    *   Перейдите в меню `Build` > `Build Bundle(s) / APK(s)` > `Build APK(s)`.
    *   После завершения сборки в правом нижнем углу появится уведомление со ссылкой "locate" — там и будет лежать ваш `app-debug.apk`.
5.  **Настройка GigaChat**:
    *   Запустите приложение на телефоне.
    *   Нажмите на иконку настроек в оверлее.
    *   Введите ваш API ключ от GigaChat.

## Использование
*   Откройте игру "Дурак" в браузере или приложении.
*   Нажмите "Запустить ассистента" в приложении Durak AI.
*   Разрешите захват экрана и отображение поверх других окон.
*   Бот начнет подсказывать ходы прямо поверх игры.

> [!TIP]
> Для лучшей точности убедитесь, что яркость экрана достаточна и карты четко видны.
